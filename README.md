## Contenido del proyecto
Carpeta `BibliotecaUI/` con:
- `index.html` — demo que muestra todos los componentes.
- `styles/variables.css` — variables CSS y reset.
- `styles/components.css` — estilos de componentes (BEM-like).
- `styles/utilities.css` — clases utilitarias.
- `scripts/main.js` — funciones JS para modal, tabs, accordion y toast.
- `README.md` — este documento.

## Componentes desarrollados
1. Botones (primario, secundario, ghost, icon, small, disabled, hover/active)
2. Tarjetas de contenido (card responsive)
3. Campos de formulario (inputs, textarea, select, checkbox, radio)
4. Navbar responsivo (menú hamburguesa)
5. Modal accesible (gestión básica de foco)
6. Tooltip / Alerts / Badges
7. Acordeón (expandible)
8. Tabs (navegación por pestañas)
9. Toast notifications

## Tecnologías y técnicas usadas
- CSS variables (`--var`) para tema y espaciados.
- Flexbox y Grid para layouts responsivos.
- Media queries para 3 puntos de quiebre.
- Pseudo-clases (`:hover`, `:active`) y pseudo-elementos (`::after`) para tooltip.
- Transiciones y sombras para microinteracciones.
- Estructura BEM-like para modularidad.
- JS vanilla para comportamiento de componentes interactivos.
